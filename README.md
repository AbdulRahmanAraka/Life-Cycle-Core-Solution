# Lifecycle Core Solutions Website

Marketing website for Lifecycle Core Solutions built with React and Vite.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create a `.env.local` with EmailJS config:
   `VITE_EMAILJS_SERVICE_ID`, `VITE_EMAILJS_TEMPLATE_ID`, `VITE_EMAILJS_PUBLIC_KEY`
3. Run the app:
   `npm run dev`
